🛠️ DigiTools - Digital Goods Marketplace

📌 Description

    DigiTools is a modern and responsive web application built with React where users can explore and purchase premium digital tools. It provides a smooth user experience with real-time cart updates and interactive UI.

🚀 Technologies Used
    React.js
    Tailwind CSS
    DaisyUI
    JavaScript (ES6+)
    React Toastify

✨ Features
🛒 Smart Cart System

   Users can add products to the cart, remove items, and see real-time updates in the navbar.

🔄 Product & Cart Toggle

   Easily switch between product listing and cart view with a smooth UI experience.

🔔 Toast Notifications

   Instant alerts for adding, removing, and checkout actions using React Toastify.
